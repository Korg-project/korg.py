pyKorg documentation
====================

.. warning::

   ``pyKorg`` is in early development!

``pyKorg`` is a python package for performing fast 1D LTE stellar spectral synthesis.
Under the hood, it uses the `Korg.jl <https://github.com/ajwheeler/Korg.jl>`__ julia package.
